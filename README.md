*****Web Development Practicals and Capstone Project*****
This repository documents the development of four core practical experiments and the final Capstone Project. The projects progress from static front-end design to complex, full-stack system architecture.

**Practical Experiments**
**Experiment 1: Personal Portfolio**
A responsive website designed to showcase professional skills and academic achievements. It focuses on semantic HTML, CSS Flexbox/Grid layouts, and mobile-first design principles.

**Experiment 2: Company Landing Page**
A business-centric landing page optimized for user conversion. This project emphasizes modern UI/UX trends, including navigation components, service sections, and professional branding.

**Experiment 3: Interactive Quiz Application**
A dynamic application built with JavaScript to handle state management. It features randomized questions, real-time score calculation, and interactive feedback based on user input.

**Experiment 4: Advanced To-Do System**
A full-stack task management tool featuring:

Database Integration: Persistent storage for user tasks.

Time Management: Automatic timestamps for task creation and completion.

Automated Notifications: Backend logic using Nodemailer to send email alerts to users before a scheduled task begins.

**Capstone Project: AuraTravels**
AuraTravels is a comprehensive travel management platform designed to streamline trip planning. The application allows users to browse destinations, book travel packages, and manage itineraries. Key features include:

User Authentication: Secure login and profile management.

Search and Filter: Advanced logic to find travel options based on budget and location.

Booking Engine: Integrated system for managing reservations and payment status.

April 2026 Submission: This project serves as the final evaluation of full-stack competency.

Tech Stack: HTML5, CSS3, JavaScript, Node.js, Express, MongoDB.
